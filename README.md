# IKT csoport munka

### Felhasznált technológiák:
- GIT
- HTML
- CSS3
- JavaScript

---

**GIT**: Egy nyílt-forráskodú, elosztott verziókezelő szoftver, vagy másképpen egy szoftverforráskód-kezelő rendszer, amely a sebességre helyezi a hangsúlyt. Minden Git munkamásolat egy teljes értékű repository teljes verziótörténettel és teljes revíziókövetési lehetőséggel, amely nem függ a hálózat elérésétől vagy központi szervertől.

**HTML**: Hyper Text Markup Language. Leíró nyelv, melyet a böngésző értelmez és a benne található információk alapján megjeleníti az oldalt. Weboldalak készítéséhez fejlesztettek ki, és mára már internetes szabvánnyá vált. 

**CSS3**: Cascading Style Sheets. Stílusleíró nyelv, HTML és más struktúrált dokumentumok megjelenését írjuk meg. Segítségével látványosabb tehetjük a megjelenését.

**JavaScript**: Objektumorientált szkriptnyelv, ami egy weboldalon történő funkciók működéséért felel.
